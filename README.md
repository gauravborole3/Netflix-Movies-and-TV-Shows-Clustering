# Netflix Movies and TV Shows Clustering

## Project Overview
This project performs exploratory data analysis and unsupervised machine learning on Netflix Movies and TV Shows data to identify meaningful content clusters.

The goal is to understand content patterns, country-wise distribution, Netflix’s focus on TV shows, and to group similar content using clustering techniques.

## Dataset
- Source: Flixable (Netflix content dataset)
- Each row represents a movie or TV show available on Netflix
- Includes numerical, categorical, and text-based features

## Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Outlier and data quality analysis
- Feature engineering and TF-IDF text processing
- Feature scaling
- Clustering using:
  - K-Means
  - Hierarchical Clustering
- Cluster interpretation and business insights

## Tools & Technologies

- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
